# 全角字符修正与特殊字符补充

- 本包添加了几个特殊字符字体，用于补充部分新确定的元素名称
- 本包修正了中文全角字符的宽度和位置问题。

## 本包建议翻译文本采用左侧书写，在打包阶段使用脚本转换成右侧字符
